学生生活で学んだ事

　高専では部活動に所属しておらず、ずっと勉学に励んできました。入学したての頃はバスケットボール部に所属したかったが、友達と新しいことを始めたいという気持ちがあり、陸上部に所属していました。ですが、勉学が疎かになってしまうので辞めてしまいました。元々運動がとても好きなので、勉学が疎かにならない程度に運動を続けたいと思って、3、4年前から部活動とは別で、社会人バスケットボールチームに所属して楽しく運動をしていました。 運動が好きという以外にも、運動を続けたい理由がありました。高専で所属していた陸上部や中学時代に所属していたバスケットボール部を辞めて、運動部に所属しているほうが、体力もつき集中力も養えるということに気が付きました。運動から離れると集中力が切れやすくなったと感じ、勉学への支障も少しあったので、週数回、社会人バスケットをすることで、集中力の低下をカバーすることができたかと思いました。

また、高専での学生生活において苦労したことの方が多くあったと思います。1、2年生では科目数が多く、すべての教科の課題や勉強をするのが大変でした。3年生になると電気や情報の専門科目が多く、要領良く勉強できずにテストで低い順位になり、悔しい思いをしました。分からない所はその日のうちに解決し、テスト前には計画を立て必ず実行するように努力をしたことが、とても印象に残っています。4年生では就職活動で精神的に追い詰められた時期もありました。高専で養うことができた、粘り強さと忍耐力、自ら考え行動することを肝に銘じ、これからの人生に生かしていきたいと思っています。

自分の教訓がありまして、「人生において、道に迷ったら、自分の苦手な道を選びなさい。そうすれば人生は成功する。」という言葉です。中学時代に通っていた塾の先生の言葉で、20年間生きてきた中で最も自分に影響のあった人と断言できます。自分を律し、自ら考え、勉学に対して全力で取り組むことができるようになったのもその方のおかげだと思っています。中学生だと意志が弱く、何かと誘惑に負けてしまいうことが多かったのですが、喝を入れてもらってから、人が変わったように勉学に励み、自分でも驚いていたことを覚えています。傲りかもしれませんが、こんな怠けていた自分でも可能性があり、まだまだ成長できるのだと思っていました。その人に会っていなければ、何事にも全力で取り組むことのできる自分がいないと思っています。かなり偏った考えの人でしたが、たくさんの知識や考えを学ぶことができました。

趣味、特技

　体を動かすことが好きで、小さい頃から習い事をさせてもらっていまして、スキーは５歳の頃からずっと続けています。競技に出るほどではないですが、並みには滑れて、昨年度からフリースタイルスキーに挑戦しています。昨年度は10回程度山に滑りに行き、グラブ（飛んだ時に板を掴む）や360（横回転のスピンで360°回転する）などができるようになりました。また、山の頂上まで登り、誰も滑っていないコース外を滑るなどバックカントリーもしています。旭川より札幌方が練習できる場所が多いと耳にしましたので、練習したいと思っています。また、中学時代にはバスケットボールをしていましたので、仕事や一人暮らしに少し慣れ、余裕ができたら何らかの社会人チームに所属するか、自主的に週何回か運動をしたいと思っております。 　また、アウトドアが好きで、夏はキャンプや釣り、海水浴によく行っていました。高専3年生になると運転免許を取得している友達とキャンプに毎年2回程度行っていました。遠軽、ニセコ、支笏湖など有名所や地元の小さなキャンプ場にも行きました。どちらにも成功や失敗がありまして、準備不足という失敗が多かったですが、何事も上手くいってしまうっていう自分達のキャンプでの定番でした。今年度は登山などのもっと様々なアウトドアに挑戦したいと思っています。

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
